现阶段网络上会存在很多盗用别人的图片、音视频等现象，这样会给原作者带来很不愉快的影响。
针对这一问题，我创建了一个项目，并用python实现了它。利用盲水印算法可以解决这一问题。
盲水印算法可以把水印嵌入到图片里面，而得到的图片在视觉上与原图几乎没有任何差别。如果对嵌入后的图片进行提取操作，也可以实现对水印的提取，这样就可以分辨谁才是原图的作者。
本项目实现了给图片嵌入一个图片类型的盲水印、给图片嵌入一个字符串的盲水印并且也实现了给音频嵌入一个字符串的盲水印。

“源代码”
    该文件夹下有四个文件：
    image.py：实现对图片嵌入字符串
    jpg.py：实现对图片嵌入图片
    music.py：实现对音频文件嵌入字符串
    ui.py：实现另外三个文件的整合及UI界面

目录下其他文件是代码运行中操作过的图片及音频文件。
